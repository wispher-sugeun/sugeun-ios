## Description
만능이니까 수근수근이에요

금방 잊어버리는 글귀, 점점 쌓여가는 수 많은 링크들, 있는지도 몰랐던 기프티콘들

누가 대신 기억해줄 순 없을까? 수근수근이에게 말해주세요! (수근수근)

글, 링크, 기프티콘까지 !! 나만의 정리함에서 톡톡히 필요한 내용들을 정리해보아요.


여태 메모장에 툭툭, 대화방에 툭툭, 급히 옮겨 적으셨다구요 ?

이제는 나만의 정리 앱 수근수근 이를 통해 효율적으로 정리해보세요 !!

## Stack
- Storyboard 기반의 UI
- MVVM 패턴 활용
- Alamofire 네트워크 통신
- Local Notification
